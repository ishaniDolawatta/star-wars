> Star wars combined search and view application using SWAPI.

### ✨ [Demo](https://google.com)

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

## Run tests

```sh
npm test
```
## Technologies

* Main tech : react, react-query, typescript,
* Styling : styled-components, scss
* Testing : Jest, Cypress.io
* Internationalisation : i18next
* Icons : react-icons
